Food Store - Carrito de Compras
================================

📌 Descripción
Este proyecto es una aplicación web que simula un carrito de compras para una tienda de alimentos.
Permite mostrar productos con nombre, descripción, imagen y precio, agregarlos o eliminarlos del carrito,
calcular subtotal y total en tiempo real, vaciar el carrito y confirmar la compra.
Los datos se guardan en localStorage para mantener el estado entre recargas.

🚀 Instrucciones para ejecutarlo
1. Clonar o descargar el proyecto.
   git clone <URL-del-repositorio>

2. Instalar dependencias con pnpm:
   pnpm install

3. Ejecutar el servidor de desarrollo:
   pnpm dev

4. Abrir el proyecto en el navegador en la dirección indicada por pnpm (por defecto http://localhost:5173).
